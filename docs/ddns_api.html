更新網域（DDNS）API 說明

本頁說明如何使用 API 來更新你的動態網域（DDNS）。

更新網域（DDNS）

  • 指定更新 IP：
    curl "https://taiwanfrp.ddns.net/api/ddns_update?username=你的帳號&password=你的密碼&domain=你的網域&ip=想更新的IP"
    直接將「你的網域」的 IP 更新為「想更新的IP」。
  • 自動更新（更新成發送請求的電腦 IP）：
    curl "https://taiwanfrp.ddns.net/api/ddns_update?username=你的帳號&password=你的密碼&domain=你的網域"

網頁版 DDNS 自動更新工具

你可以使用圖形化的 DDNS 自動更新工具，定期自動將你的網域 IP 更新到最新。

  • 填寫 帳號、密碼、網域（預設自動補 .taiwanfrp.me），IP 可選填（不填則自動偵測）。
  • 可選擇自動更新的間隔時間（1~60分鐘）。
  • 按下「立即更新」可手動觸發一次更新。
  • 每次更新後會顯示伺服器回應與結果。
